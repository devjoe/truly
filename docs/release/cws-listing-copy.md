## Name

Truly

## Short Description

Local-first reading clarity for social feeds and web pages.

## Full Description

Truly helps readers slow down and inspect information quality before reacting.
It adds a compact reading hint near supported posts and a user-opened side panel
with summary, context, follow-up questions, language-convention checks, and
manual handoff to external tools.

Truly is local-first. Reading analysis runs in the model environment selected by
the user: Chrome built-in Gemini Nano when available, a local model endpoint, or
a private endpoint the user configures. Truly does not operate a project-owned
backend for feed content and does not include product analytics or telemetry.

External tool use is manual. Search, Meta AI handoff, Markdown copy, and
Markdown download happen only when the user clicks the relevant action.

Early preview scope: supported social feed surfaces may change as websites
update their layout. Model quality and speed depend on the selected local or
private model. Truly should be used as reading assistance, not as authoritative
truth.

Truly is not an ad blocker, automatic fact-checker, moderation tool, scraping
service, or account automation tool.
