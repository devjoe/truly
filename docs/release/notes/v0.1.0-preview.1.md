# Truly 0.1.0 Preview 1

This is an early preview for friends and testers who are comfortable installing
an unpacked Chrome extension.

## Install

1. Download `truly-extension-0.1.0-677724d22004.zip` from the assets below.
2. Unzip the file.
3. Open `chrome://extensions` in desktop Chrome.
4. Enable **Developer mode**.
5. Choose **Load unpacked**.
6. Select the unzipped extension folder.

Do not install the source zip unless you want to inspect or build the code.

## Set Up

After installation, open the extension Options page.

- Start with Chrome built-in Gemini Nano when it is available.
- Use Ollama or an OpenAI-compatible endpoint if you already run local or
  private models.

Model setup guide: <https://github.com/devjoe/truly/blob/main/docs/model-setup.md>

## Try

- Open a supported page in desktop Chrome.
- Check whether reading hints appear before posts.
- Open the expanded view and side panel.
- Report confusing, inaccurate, or broken behavior.

Feedback form: <https://trulyreader.org/feedback/>

## Notes

- This is not a Chrome Web Store install yet.
- It does not auto-update. Reinstall a newer preview when one is released.
- Site layouts and model availability can affect behavior.
